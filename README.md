# climate-challenge-week0
# African Climate Trend Analysis - Week 0

 Project Overview
This repository contains the initial setup and Exploratory Data Analysis (EDA) for analyzing climate trends in five African countries using NASA POWER data (2015-2026).

 Project Structure
- `notebooks/`: Jupyter notebooks for data analysis.
- `src/`: Source code for the project.
- `scripts/`: Python scripts for data cleaning and processing.
- `tests/`: Unit tests to ensure code quality.
- `.github/workflows/`: CI/CD pipeline using GitHub Actions.

 Setup Instructions
1. Clone the repository:
   `git clone https://github.com`
2. Install dependencies:
   `pip install -r requirements.txt`
